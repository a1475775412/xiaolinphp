# 功能
- View (视图/模板渲染)
- Hook (钩子)
- ErrorManager (错误管理)
- Router (路由)
- Language (多语言支持)
- Controller (控制器)

- Autoload (class自动加载)
- AntiCSRF (formhash,防止CSRF攻击)
- PerformanceStatistics (性能统计)
- Cache (缓存)

- Safe (安全处理函数)
- Config (配置文件类)
- DB (数据库类)
